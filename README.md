## 描述/describe
依赖于PyQt5的简易桌面数字时钟，点击脚本即可运行。  
本人写这个脚本是为了在学习时候可以置顶一个时钟，方便的计算时间，在linux上找了一圈没有找到合适的，要么不是图形界面，要么太小不方便看时间。  
kde可以在任务栏选择置顶工具。   

This is a simple desktop digital clock that uses PyQt5; simply click the script to run it.   
I wrote this script to keep a clock on top of my screen while studying, making it easy to calculate time. I searched extensively on Linux but couldn't find a suitable one; they were either not graphical or too small to easily read the time.   
In KDE, you can select "Place on Top" in the taskbar.

## 自定义/customization
颜色：修改 "color: #00FFCC;" 中的色值。  
大小：调整 setFixedSize(300, 120) 和字体大小 48。  
透明度：修改 QColor(0, 0, 0, 180) 的最后一个参数。   

Color: Modify the color value in "color: #00FFCC;".   
Size: Adjust setFixedSize(300, 120) and font size to 48.   
Opacity: Modify the last parameter of QColor(0, 0, 0, 180).
